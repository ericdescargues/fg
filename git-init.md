# git init
## Introduction

La commande init permet d'initialiser la gestion de configuration dans un répertoire.
Un répertoire .git est crée contenant la structure initiale d'un répertoire git.

## Commande

```bash
git init
```

## Ajout de fichiers

```bash
git add --all
```

## 1er commit

```
git -m "First commit"
```
